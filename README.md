# Loan_Analysis
An exploratory data analysis and classification prediction of Kaggle [LendingClub.com Loan Dataset](https://www.kaggle.com/itssuru/loan-data)

# Final Results (Accuracy)
- **RFC** - 83.51%
- **DTC** - 73.61%
- **XGB** - 82.55%
